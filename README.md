# RIA-Quotation-management-system

Project for "Tecnologie informatiche per il web" course of Politecnico di Milano.

This project is a **rich internet web application** (**single-page** web application), if you want to check out the **pure HTML** version check [this repo](https://github.com/FedericoMainettiGambera/Quotation-management-system-pureHTML).

## What is this project about?

A web application that allows the management of quotes for customized products.

This project covers topics like:
- Backend and frontend development;
- pure HTML application
- RIA (rich internet application)
- authentication & sessions
- Java and TOMCAT
- database (MySQL)

and many more!
